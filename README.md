# 🚀 Overview

IPO Risk Engine is a Data Science and Analytics platform built to analyze large-scale structured datasets through:

- 📊 Exploratory Data Analysis (EDA)
- 📈 Statistical Analysis
- 🧠 Feature Engineering
- 📉 Correlation Modeling
- ⚡ Interactive Visualization

The project focuses on extracting meaningful patterns, identifying behavioral trends, and building data-driven insights using Python-based analytics workflows.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/amogh2222/IPORiskEngine.git
cd IPORiskEngine
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Dashboard

```bash
streamlit run app.py
```
